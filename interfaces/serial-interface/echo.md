# Echo

```sh
sudo screen /dev/ttyUSB0 115200 # usually the fastest you can go
```

{% hint style="warning" %}

Use `ls -l /dev | grep ttyUSB0` to make sure that `/dev/ttyUSB0` exists.

{% endhint %}
